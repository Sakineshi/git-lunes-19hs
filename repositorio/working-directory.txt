WORKING DIRECTORY:
	Es la carpeta donde se lleva a cabo el trabajo
	de nuestro proyeto, generalmente utilizaremos
	las herramientas de desarrollo elegidas por
	nosotros y muy rara vez la consola.

COMANDOS:

CD (check directory)
	Nos permite navegar entre los directorios
	de la computadora (carpetas) utilizando
	diferentes variantes.
	
	cd .. => vuelve al directorio anterior
	cd <folder> => abre el directorio especificado
	cd \ => abre el directorio raiz (C:)

mkdir (make directory)
	Permite crear directorios con los nombres
	que especifiquemos despues de comando
	
	mkdir <folder> <folder>
